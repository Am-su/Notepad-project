## Description

제목과 내용으로 이루어진 메모를 추가/삭제/저장 할 수 있는 앱으로 회원가입/로그인/로그아웃/비밀번호 찾기를 구현했습니다.
웹 호스팅과 데이터베이스는 firebase를 이용했습니다.
